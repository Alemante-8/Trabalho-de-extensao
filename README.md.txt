# Classy

Classy é um aplicativo de agendamento de consultas online, focado em permitir que usuários escolham datas e horários específicos para agendar compromissos.

## Funcionalidades

- Escolher data e horário para consultas.
- Adicionar informações do cliente, tipo de consulta e notas.

## Estrutura do Projeto

- `MainActivity.java`: Tela principal com calendário.
- `ScheduleActivity.java`: Tela para agendamento de consultas.
- `database/`: Contém as classes do banco de dados (Room).
- `res/layout`: Arquivos XML de layout para as telas principais.

## Tecnologias Usadas

- Android SDK
- Java
- Room Database
